J is a text editor for programmers, available under the GNU General Public License.

Compatibility[edit]

J is written in Java and runs on Linux, Mac OS X, OS/2, Unix, VMS, and Windows. It is part of the ArmedBear Common Lisp (ABCL) project on SourceForge.
