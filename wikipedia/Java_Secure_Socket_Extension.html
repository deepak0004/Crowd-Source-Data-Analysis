In computing, the Java Secure Socket Extension (JSSE) provides a set of packages which enable secure Internet communications. It implements a Java technology version of the Secure Sockets Layer (SSL) and the Transport Layer Security (TLS) protocols. It includes functionality for data encryption,[1] server authentication, message integrity, and optional client-authentication.

JSSE, an optional package for Java versions 1.2 and 1.3, became integrated into version 1.4[clarify].
