JEngine is an open source, standards-compliant, enterprise integration engine implemented in Java. The server framework, based on Java Platform, Enterprise Edition (Java EE), provides a robust standard for deploying a multi-tier enterprise integration engine. By utilizing the Java EE standard, a complete framework of standardized components, services, and interfaces will be developed while maintaining the promise of portability across operating systems.
