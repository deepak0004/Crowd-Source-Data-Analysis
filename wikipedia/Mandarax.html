Mandarax is an open source Java class library for deduction rules. It provides an infrastructure for defining, managing and querying rule bases.

Mandarax has an expressive and simple domain-specific language to define relationships through derivation rules. Many Java language elements are re-used in the Mandarax language. It also has a modular, template-based compiler, as well as support for advanced features such as negation as failure, In Domain expressions, relationship inheritance and aggregations.[1]
